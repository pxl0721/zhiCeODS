# 智能体赋能的企业运营分析与决策支持系统

## 📌 项目简介

本项目是中国大学生计算机设计大赛参赛作品，旨在构建一个基于大模型智能体的企业运营分析与决策支持系统。系统通过自然语言交互，让用户能够像与分析师对话一样查询企业运营数据，并自动获得数据洞察与决策建议，实现从“看数据”到“问决策”的体验升级。

**核心亮点**：引入 LangChain 多智能体框架，结合阿里云通义千问大模型，实现自然语言自动转化为数据分析代码、智能归因异常波动、自动生成运营报告等功能。

## ✨ 主要功能

- **📊 运营仪表盘**：实时展示核心 KPI（总销售额、总利润、平均利润率）及趋势图表。
- **💬 智能问答**：支持自然语言查询，例如“上个月哪个产品利润最高？”，系统自动分析并返回文字答案。
- **📈 动态图表**：根据问题类型自动切换图表（折线图/柱状图），直观呈现分析结果。
- **📁 自定义数据上传**：支持用户上传 CSV 格式的运营数据，系统自动解析并切换数据源，智能体同步适配。
- **🤖 智能体决策支持**：基于查询结果，AI 自动生成业务解读与决策建议（不超过100字）。
