二、快速运行指南（推荐方式）
1. 进入部署包
打开 部署包 文件夹，进入 智策运营决策系统 子文件夹。

2. 启动应用程序
双击 智策运营决策系统.exe。

3. 访问系统
程序启动后，将自动打开您的默认浏览器并跳转至系统首页（http://127.0.0.1:5000）。您也可以通过浏览器手动访问该地址。

⚠️ 注意：启动过程中会出现一个命令行窗口，请勿关闭，它是程序运行的服务窗口。如需停止服务，可直接关闭该窗口。

4. 体验完整功能
系统内置了模拟数据，您可以：

在仪表盘查看核心KPI和趋势图表。

使用右侧聊天框进行自然语言查询（例如“上个月哪个产品利润最高？”）。

通过顶部导航栏访问“运营评估”、“风险机会”、“生成报告”等AI分析功能。

点击“导入CSV数据”上传自定义数据集（格式需符合模板要求）。

三、从源码运行（如需查看代码或二次开发）
若评委老师希望从源码直接运行，请按以下步骤操作：

环境要求：Python 3.11（推荐），或已安装 Miniconda/Anaconda。

1. 进入源码目录

bash
cd 源码
2. 创建并激活虚拟环境（可选）

bash
# 使用 venv
python -m venv .venv
# Windows 激活
.venv\Scripts\activate
# macOS/Linux 激活
source .venv/bin/activate
3. 安装依赖

bash
pip install -r requirements.txt
4. 配置 API 密钥
在 源码/ 目录下，将 .env.example 复制为 .env，并填写您的阿里云百炼 API Key：

text
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
DEMO_MODE=false
如无API Key，可将 DEMO_MODE 设为 true，系统将使用模拟数据演示（不调用真实AI）。

5. 启动应用

bash
python app.py
浏览器访问 http://127.0.0.1:5000 即可。

四、常见问题
问题	解决方案
双击exe后命令行窗口闪退	请检查 部署包/ 文件夹内是否包含 .env 文件（已内置），若缺失可手动从源码复制。
网页无法打开	确认命令行窗口显示 Running on http://127.0.0.1:5000，并检查防火墙是否拦截。
提问时返回403错误	部署包中的 .env 文件已配置有效的演示密钥，若密钥过期，请联系作者。
想关闭服务	在命令行窗口中按 Ctrl+C，或直接关闭窗口。
